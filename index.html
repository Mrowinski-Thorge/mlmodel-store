Eine kuratierte Sammlung von Core ML-Modellen, bereit zum Download für Ihre Apple-Plattform-Apps.

Was sind .mlmodel Dateien?

Optimiert für Apple-Geräte

.mlmodel ist das Format für Core ML. Modelle in diesem Format laufen extrem schnell und effizient auf iPhone, iPad und Mac.

Datenschutz im Fokus

Da die Modelle direkt auf dem Gerät des Nutzers ausgeführt werden, bleiben alle Daten privat. Es ist keine Serververbindung nötig.

Einfache Integration

Entwickler können .mlmodel-Dateien mit nur wenigen Zeilen Code in ihre Xcode-Projekte integrieren.
